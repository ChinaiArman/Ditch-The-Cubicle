Help and Documentation

A little confused about what each of the pages does? Discover what the homepage, the map page, and the profile page offers.

Home

The home page displays a wide cataglouge of events and attractions. You can bookmark events you are interested in attending. You can also click onto the event card for additional information about the event. Once inside the event details page, the event information is displayed, a map of where the event is located, and a section to review and write comments about the event.

Go to Home

Map

The map page shows event locations in a map view. You can use this page to find events located within specific neighbourhoods, or events that are closest to you. Zoom in to see exactly where events are located and zoom out to see events located further away. Try clicking on the pin icon in the map to view event details.

Go to Map

Profile

The profile page displays information specific to your account. Here, you can view events you have saved, edit your name and profile picture, and customize the background of your webpage. The profile page can only be accessed once you have logged into your account. Make sure to sign up if you haven't already!

Go to Profile
